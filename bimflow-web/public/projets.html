📁 Projets BIMFlow

Chaque projet Revit est identifié par un code unique. Cliquez sur un projet pour l'ouvrir, ou entrez un code partagé par votre équipe.

⚙ Comment configurer un projet Revit
Option A — Sans recompilation (maintenant) :
Dans Revit → Gérer → Informations du projet → N° de projet, entrez un code court unique (ex : VLM-001). Ce code sera utilisé comme identifiant de projet.

Option B — Après recompilation du plugin (permanent) :
Le plugin génère automatiquement un code à partir du GUID du fichier Revit. Aucune configuration nécessaire.
